What is GitHub?

GitHub is mostly used for code sharing and publishing. The site allows users to upload and create projects that can be viewed and added to by other people. Programmers also use GitHub as a social networking site. It is a great place for people to enhance their programming skills. You can create a personal or host other websites using GitHub pages.

Creating Your Own Website

To create your personal website using GitHub, you should follow the steps listed below.

  1. First, you need to create a repository. It has to be username.github.io (use your username).
  2. Next, if you are using Windows, you need to publish and clone the repository.
  3. Open your favorite text editor (such as Notepad ++) and create an index.html file.
  4. Finally you have to enter the repository, commit your changes, and press the sync button.
Your website will sync in a few minutes and it will show up at http://username.github.io/.

There are a few thing that you need to know to better understand how to use GitHub.

  • Repositories are used to store your projects. There are public and private repositories. Privates repositories are not free so it may be easier to just stick to public.

  • A commit is a snapshot of the files that are in your project at a certain time.

  • README's are used like comments. They can tell a user how to download something or just give an explaination.

  • A fork is a copy of a repository. Forking allows you to experiment with projects and make changes without affecting the original project.

  • You can also use GitHub for social reasons. You can follow other people, watch projects, and contribute to projects that allow for it.

Once you have successfully created you index.html file, you can edit it as much as you want. To do this you first need to select and open the index file.
On the toolbar above, press the edit button (it looks like a pen). It will open up to you code in an editable form. When you're done, scroll down and save your changes.
